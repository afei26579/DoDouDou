## 1. 架构设计

### 1.1 整体架构原则

| 原则 | 具体要求 | 原因 |
|------|----------|------|
| API-First | 前后端完全分离，后端只提供 RESTful API | 同一套API服务Web/H5/小程序/App |
| 后端语言统一 Python | 所有后端服务使用 Python | 团队技能统一，图像处理生态成熟 |
| 前端响应式优先 | 一套前端代码适配手机/平板/桌面 | 降低维护成本 |
| 核心计算可选端侧 | 转图引擎同时提供服务端版本和WebAssembly版本 | 隐私+速度+降低服务器成本 |
| 无状态服务 | 后端服务无状态，状态存储在数据库/缓存 | 便于水平扩展 |
| 渐进式架构 | MVP用单体应用，预留拆分接口 | 避免过度设计 |

### 1.3 多平台适配策略

| 阶段 | 平台 | 前端方案 | 后端方案 |
|------|------|----------|----------|
| Phase 1 (MVP) | 移动端H5 + 桌面Web | React响应式单页应用 | FastAPI RESTful API |
| Phase 2 | 微信小程序 | Taro / uni-app 跨端框架，或原生小程序 | **同一套API**，新增微信登录/支付接口 |
| Phase 2 | 抖音小程序 | Taro 跨端编译 | **同一套API**，新增抖音登录接口 |
| Phase 3 | iOS / Android App | React Native 或 Flutter | **同一套API**，新增推送/IAP接口 |

**关键设计：后端API完全平台无关，通过请求头 `X-Platform` 区分来源，仅在认证、支付等平台相关逻辑中做差异处理。**

## 2. 技术选型详解

### 2.1 后端技术栈

| 组件 | 选型 | 版本 | 选型理由 |
|------|------|------|----------|
| 语言 | Python | 3.11+ | 团队主语言；图像处理库生态最强(Pillow/OpenCV/NumPy/scikit-image) |
| Web框架 | FastAPI | 0.110+ | 异步高性能；自动OpenAPI文档；类型安全；Python生态最现代的框架 |
| ORM | SQLAlchemy 2.0 + Alembic | 2.0+ | 成熟稳定；支持异步；Alembic管理数据库迁移 |
| 异步运行时 | uvicorn + asyncio | — | FastAPI标配 |
| 任务队列 | Celery + Redis | 5.3+ | 转图等耗时任务异步处理；MVP阶段可选用 FastAPI BackgroundTasks 替代 |
| 图像处理 | Pillow + NumPy + scikit-image | — | 核心转图引擎依赖 |
| 色彩计算 | colormath / colour-science | — | CIEDE2000 色差计算 |
| PDF生成 | reportlab / WeasyPrint | — | 打印图纸PDF导出 |
| 认证 | PyJWT + bcrypt | — | JWT Token认证 |
| 数据校验 | Pydantic v2 | — | FastAPI原生集成 |

**grid_data 结构说明：**

```jsonc
// grid_data 存储每个像素点的颜色ID
// 方案一：二维数组 (直观，适合中小图)
{
    "format": "2d_array",
    "data": [
        ["C01", "C01", "C05", "C05", ...],  // 第1行
        ["C01", "C19", "C05", "C05", ...],  // 第2行
        ...
    ]
}

// 方案二：游程编码 (大图压缩存储)
{
    "format": "rle",
    "data": [
        {"row": 0, "runs": [{"color": "C01", "count": 2}, {"color": "C05", "count": 27}]},
        ...
    ]
}
```

MVP 阶段使用方案一（二维数组），图较小时 JSON 大小在可接受范围（29×29=841格，约 5-10KB）。大图场景后续可切换 RLE 编码。

### 3.4 转图引擎详细设计

这是产品的核心技术壁垒，需要单独详细设计。

#### 3.4.1 处理流程

```
输入图片
  │
  ▼
① 预处理
  · 解码图片 → NumPy数组 (H × W × 3, RGB)
  · 裁剪 (根据用户裁剪参数)
  · 方向校正 (EXIF)
  · 尺寸归一化 (长边不超过2000px)
  │
  ▼
② 像素化 (Pixelizer)
  · 将图片缩放到目标网格尺寸 (如 29×29)
  · 缩放算法：LANCZOS (保留细节) 
  · 输出：小尺寸RGB图 (29×29×3)
  │
  ▼
③ 减色 (ColorReducer)
  · 将颜色数量减少到目标色数
  · 算法：改进的中位切分 + K-means精调
  · 在CIELAB色彩空间中操作 (感知均匀)
  · 输出：减色后的RGB图 + 减色色板
  │
  ▼
④ 色板匹配 (PaletteMatcher)
  · 将减色色板映射到实际豆子色板
  · 使用 CIEDE2000 色差公式找最近色
  · 避免同一颜色被重复映射
  · 输出：每个像素对应一个色板颜色ID
  │
  ▼
⑤ 后处理优化 (Optimizer)
  · 杂色清理：消除孤立异色点
  · 边缘优化：强化主体轮廓
  · 色数再检查：如果实际色数超标，合并最相近的两色
  · 输出：最终图稿数据
  │
  ▼
⑥ 大图拆板 (BoardSplitter)
  · 如果尺寸超过单板，计算拆板方案
  · 生成拆板信息 (每板的偏移量和边界)
  │
  ▼
⑦ 元数据生成
  · 统计各颜色使用数量
  · 计算难度评分
  · 估算制作时间
  · 计算适合度评分
  │
  ▼
⑧ 生成预览图
  · 用实际色板颜色渲染预览图
  · 输出到OSS
  │
  ▼
输出：方案数据 (grid_data + color_summary + metadata)
```

#### 3.4.2 核心算法实现

**像素化模块 (pixelizer.py)**

```python
# 核心逻辑伪代码

import numpy as np
from PIL import Image

class Pixelizer:
    """将图片像素化到目标网格尺寸"""
    
    # 三种方案的尺寸配置
    SCHEME_CONFIGS = {
        "simple": {"max_size": 15, "max_colors": 8},
        "standard": {"max_size": 29, "max_colors": 15},
        "fine": {"max_size": 58, "max_colors": 25},
    }
    
    def pixelize(
        self, 
        image: np.ndarray,     # (H, W, 3) RGB
        target_size: int,       # 目标短边格数
        keep_aspect: bool = True
    ) -> np.ndarray:
        """
        将图片缩放到目标网格尺寸
        
        Args:
            image: 输入图片 RGB numpy数组
            target_size: 目标短边像素数（即格数）
            keep_aspect: 是否保持宽高比
            
        Returns:
            缩放后的小图 (target_h, target_w, 3)
        """
        h, w = image.shape[:2]
        
        if keep_aspect:
            if h <= w:
                target_h = target_size
                target_w = round(target_size * w / h)
            else:
                target_w = target_size
                target_h = round(target_size * h / w)
        else:
            target_h = target_w = target_size
        
        # 使用 LANCZOS 插值缩放 (保留细节最好)
        pil_img = Image.fromarray(image)
        pil_img = pil_img.resize((target_w, target_h), Image.LANCZOS)
        
        return np.array(pil_img)
    
    def generate_schemes(
        self, 
        image: np.ndarray
    ) -> dict:
        """生成三种方案的像素化结果"""
        schemes = {}
        for scheme_type, config in self.SCHEME_CONFIGS.items():
            pixelized = self.pixelize(image, config["max_size"])
            schemes[scheme_type] = {
                "pixelized_image": pixelized,
                "max_colors": config["max_colors"],
                "grid_width": pixelized.shape[1],
                "grid_height": pixelized.shape[0],
            }
        return schemes
```

**减色模块 (color_reducer.py)**

```python
import numpy as np
from sklearn.cluster import KMeans
from skimage.color import rgb2lab, lab2rgb

class ColorReducer:
    """智能减色：将图片颜色减少到指定数量"""
    
    def reduce(
        self,
        image: np.ndarray,      # (H, W, 3) RGB, 0-255
        target_colors: int,
        method: str = "kmeans_lab"
    ) -> tuple[np.ndarray, np.ndarray]:
        """
        减色处理
        
        Args:
            image: 输入RGB图片
            target_colors: 目标颜色数
            method: 算法选择
            
        Returns:
            (reduced_image, palette)
            reduced_image: 减色后的图片 (H, W, 3)
            palette: 减色色板 (N, 3) RGB
        """
        h, w = image.shape[:2]
        
        # 转换到 CIELAB 色彩空间 (感知均匀)
        image_float = image.astype(np.float64) / 255.0
        image_lab = rgb2lab(image_float)
        
        # 展平为像素列表
        pixels_lab = image_lab.reshape(-1, 3)
        
        # K-means 聚类
        kmeans = KMeans(
            n_clusters=target_colors,
            n_init=10,
            max_iter=300,
            random_state=42
        )
        labels = kmeans.fit_predict(pixels_lab)
        centers_lab = kmeans.cluster_centers_
        
        # 转回 RGB
        centers_rgb_float = lab2rgb(
            centers_lab.reshape(1, -1, 3)
        ).reshape(-1, 3)
        centers_rgb = (centers_rgb_float * 255).clip(0, 255).astype(np.uint8)
        
        # 用聚类中心重建图片
        reduced_pixels = centers_rgb[labels]
        reduced_image = reduced_pixels.reshape(h, w, 3)
        
        return reduced_image, centers_rgb
```

**色板匹配模块 (palette_matcher.py)**

```python
class PaletteMatcher:
    """将减色后的颜色映射到实际豆子色板"""
    
    def __init__(self, palette_colors: list[dict]):
        """
        Args:
            palette_colors: 色板颜色列表
                [{"id": "C01", "lab": [100.0, 0.0, 0.0], "rgb": [255,255,255]}, ...]
        """

        self.palette_lab = np.array([c["lab"] for c in palette_colors])
    
    def match(
        self,
        reduced_palette: np.ndarray,    # (N, 3) RGB
        image_lab: np.ndarray,          # (H, W, 3) LAB
        labels: np.ndarray              # (H*W,) 聚类标签
    ) -> dict:
        """
        将减色色板映射到豆子色板
        
        Returns:
            {
                "mapping": {0: "C01", 1: "C05", ...},  # 聚类标签 → 色板ID
                "grid_data": [[...], ...],               # 最终图稿
                "color_summary": [...]                    # 颜色统计
            }
        """
        mapping = {}
        used_palette_ids = set()
        
        # 将减色色板的每个颜色转为LAB
        reduced_lab = self._rgb_to_lab_batch(reduced_palette)
        
        # 按像素数量排序，优先匹配主要颜色
        cluster_sizes = np.bincount(labels)
        sorted_clusters = np.argsort(-cluster_sizes)
        
        for cluster_idx in sorted_clusters:
            color_lab = reduced_lab[cluster_idx]
            
            # 计算与所有可用色板颜色的 CIEDE2000 色差
            best_match = None
            best_delta = float("inf")
            
            for i, pal_color in enumerate(self.palette):
                if pal_color["id"] in used_palette_ids:
                    continue  # 避免重复映射
                    
                delta = self._ciede2000(color_lab, self.palette_lab[i])
                if delta < best_delta:
                    best_delta = delta
                    best_match = pal_color["id"]
            
            mapping[int(cluster_idx)] = best_match
            used_palette_ids.add(best_match)
        
        return mapping
    
    def _ciede2000(self, lab1: np.ndarray, lab2: np.ndarray) -> float:
        """计算 CIEDE2000 色差"""
        # 使用 colormath 库或自行实现
        c1 = LabColor(lab1[0], lab1[1], lab1[2])
        c2 = LabColor(lab2[0], lab2[1], lab2[2])
        return delta_e_cie2000(c1, c2)
```

**杂色清理模块 (optimizer.py)**

```python
import numpy as np
from collections import Counter

class GridOptimizer:
    """图稿后处理优化"""
    
    def remove_noise(
        self,
        grid: list[list[str]],   # 二维颜色ID数组
        min_cluster_size: int = 3
    ) -> list[list[str]]:
        """
        移除孤立杂色点
        
        规则：如果一个颜色点的相邻8格中，
        同色点数量 < min_cluster_size，则替换为周围最多的颜色
        """
        h = len(grid)
        w = len(grid[0])
        result = [row[:] for row in grid]  # 深拷贝
        
        for y in range(h):
            for x in range(w):
                current = grid[y][x]
                neighbors = self._get_neighbors(grid, x, y, w, h)
                
                same_count = sum(1 for n in neighbors if n == current)
                
                if same_count < min_cluster_size - 1:
                    # 替换为周围最常见的颜色
                    if neighbors:
                        counter = Counter(neighbors)
                        most_common = counter.most_common(1)[0][0]
                        result[y][x] = most_common
        
        return result
    
    def merge_similar_colors(
        self,
        grid: list[list[str]],
        palette_colors: dict,      # {color_id: lab_values}
        max_delta: float = 8.0     # CIEDE2000 色差阈值
    ) -> list[list[str]]:
        """
        合并相近颜色
        
        找到色差最小的两个颜色，如果色差 < max_delta，
        将使用量少的合并到使用量多的
        """
        # 统计各颜色使用量
        color_counts = Counter()
        for row in grid:
            color_counts.update(row)
        
        used_colors = list(color_counts.keys())
        
        # 计算所有颜色对的色差
        merge_pairs = []
        for i in range(len(used_colors)):
            for j in range(i + 1, len(used_colors)):
                c1, c2 = used_colors[i], used_colors[j]
                delta = self._ciede2000(
                    palette_colors[c1], palette_colors[c2]
                )
                if delta < max_delta:
                    merge_pairs.append((c1, c2, delta))
        
        # 按色差排序，从最相近的开始合并
        merge_pairs.sort(key=lambda x: x[2])
        
        result = [row[:] for row in grid]
        for c1, c2, delta in merge_pairs:
            # 将使用量少的合并到使用量多的
            keep = c1 if color_counts[c1] >= color_counts[c2] else c2
            remove = c2 if keep == c1 else c1
            
            for y in range(len(result)):
                for x in range(len(result[0])):
                    if result[y][x] == remove:
                        result[y][x] = keep
        
        return result
    
    def _get_neighbors(self, grid, x, y, w, h):
        """获取8邻域颜色"""
        neighbors = []
        for dy in [-1, 0, 1]:
            for dx in [-1, 0, 1]:
                if dx == 0 and dy == 0:
                    continue
                nx, ny = x + dx, y + dy
                if 0 <= nx < w and 0 <= ny < h:
                    neighbors.append(grid[ny][nx])
        return neighbors
```

**跟做步骤生成 (follow_guide.py)**

```python
from collections import defaultdict

class FollowGuideGenerator:
    """生成跟做步骤数据"""
    
    def generate_steps(
        self,
        grid: list[list[str]],
        mode: str = "by_color",
        color_summary: list[dict] = None
    ) -> list[dict]:
        """
        根据引导模式生成步骤
        
        Args:
            grid: 二维颜色ID数组
            mode: by_color / by_region / by_row
            color_summary: 颜色统计信息
        """
        if mode == "by_color":
            return self._by_color(grid, color_summary)
        elif mode == "by_region":
            return self._by_region(grid)
        elif mode == "by_row":
            return self._by_row(grid)
    
    def _by_color(self, grid, color_summary):
        """按颜色引导：每种颜色一个步骤，按用量从多到少"""
        h, w = len(grid), len(grid[0])
        
        # 按颜色收集位置
        color_positions = defaultdict(list)
        for y in range(h):
            for x in range(w):
                color_id = grid[y][x]
                if color_id:  # 排除空位
                    color_positions[color_id].append([x, y])
        
        # 按用量排序（多的先放）
        sorted_colors = sorted(
            color_positions.items(),
            key=lambda x: len(x[1]),
            reverse=True
        )
        
        # 用 color_summary 获取颜色名称和hex
        color_info = {c["color_id"]: c for c in (color_summary or [])}
        
        steps = []
        for i, (color_id, positions) in enumerate(sorted_colors):
            info = color_info.get(color_id, {})
            steps.append({
                "step": i + 1,
                "type": "color",
                "color_id": color_id,
                "color_name": info.get("color_name", color_id),
                "color_hex": info.get("hex", "#000000"),
                "bead_count": len(positions),
                "highlight_positions": positions,
                "is_completed": False,
            })
        
        return steps
    
    def _by_region(self, grid):
        """按区域引导：将图稿等分为若干区块"""
        h, w = len(grid), len(grid[0])
        
        # 根据总尺寸决定分区数
        if max(h, w) <= 15:
            regions_per_side = 2   # 2×2 = 4区
        elif max(h, w) <= 29:
            regions_per_side = 3   # 3×3 = 9区
        else:
            regions_per_side = 4   # 4×4 = 16区
        
        region_h = h // regions_per_side
        region_w = w // regions_per_side
        
        steps = []
        step_num = 0
        
        for ry in range(regions_per_side):
            for rx in range(regions_per_side):
                step_num += 1
                y_start = ry * region_h
                y_end = h if ry == regions_per_side - 1 else (ry + 1) * region_h
                x_start = rx * region_w
                x_end = w if rx == regions_per_side - 1 else (rx + 1) * region_w
                
                positions = []
                color_counts = defaultdict(int)
                
                for y in range(y_start, y_end):
                    for x in range(x_start, x_end):
                        if grid[y][x]:
                            positions.append([x, y])
                            color_counts[grid[y][x]] += 1
                
                steps.append({
                    "step": step_num,
                    "type": "region",
                    "region_label": f"区域 {step_num}",
                    "region_bounds": {
                        "x": x_start, "y": y_start,
                        "w": x_end - x_start, "h": y_end - y_start
                    },
                    "bead_count": len(positions),
                    "highlight_positions": positions,
                    "color_breakdown": dict(color_counts),
                    "is_completed": False,
                })
        
        return steps
    
    def _by_row(self, grid):
        """按行引导：逐行完成"""
        h, w = len(grid), len(grid[0])
        steps = []
        
        for y in range(h):
            positions = []
            for x in range(w):
                if grid[y][x]:
                    positions.append([x, y])
            
            if positions:
                steps.append({
                    "step": y + 1,
                    "type": "row",
                    "row_index": y,
                    "bead_count": len(positions),
                    "highlight_positions": positions,
                    "is_completed": False,
                })
        
        return steps
```

#### 3.4.3 方案生成器 (scheme_generator.py)

```python
class SchemeGenerator:
    """三方案生成器：协调各模块完成端到端转图"""
    
    def __init__(self):
        self.pixelizer = Pixelizer()
        self.color_reducer = ColorReducer()
        self.palette_matcher = PaletteMatcher()
        self.optimizer = GridOptimizer()
        self.board_splitter = BoardSplitter()
    
    async def generate(
        self,
        image: np.ndarray,
        palette: list[dict],
        crop_params: dict = None
    ) -> list[dict]:
        """
        生成三种方案
        
        Returns: 包含 simple/standard/fine 三个方案的列表
        """
        # 1. 预处理
        if crop_params:
            image = self._crop(image, crop_params)
        
        # 2. 计算适合度
        suitability = self._evaluate_suitability(image)
        
        # 3. 对每种方案执行完整流程
        schemes = []
        for scheme_type, config in Pixelizer.SCHEME_CONFIGS.items():
            scheme = await self._generate_single_scheme(
                image, palette, scheme_type, config
            )
            schemes.append(scheme)
        
        return schemes, suitability
    
    async def _generate_single_scheme(
        self, image, palette, scheme_type, config
    ):
        # Step 1: 像素化
        pixelized = self.pixelizer.pixelize(image, config["max_size"])
        
        # Step 2: 减色
        reduced, reduced_palette = self.color_reducer.reduce(
            pixelized, config["max_colors"]
        )
        
        # Step 3: 色板匹配
        match_result = self.palette_matcher.match(
            reduced_palette, pixelized, palette
        )
        
        # Step 4: 生成网格数据
        grid_data = match_result["grid_data"]
        
        # Step 5: 后处理优化
        grid_data = self.optimizer.remove_noise(grid_data)
        
        # Step 6: 检查是否需要拆板
        board_layout = None
        grid_h, grid_w = len(grid_data), len(grid_data[0])
        if grid_h > 29 or grid_w > 29:
            board_layout = self.board_splitter.split(grid_w, grid_h)
        
        # Step 7: 统计
        color_summary = self._compute_color_summary(grid_data, palette)
        total_beads = sum(c["count"] for c in color_summary)
        
        # Step 8: 生成预览图并上传
        preview_url = await self._render_and_upload_preview(
            grid_data, palette, scheme_type
        )
        
        return {
            "type": scheme_type,
            "label": {"simple": "简单版", "standard": "标准版", "fine": "精细版"}[scheme_type],
            "tag": {"simple": "推荐新手", "standard": "推荐", "fine": "效果最佳"}[scheme_type],
            "grid_width": grid_w,
            "grid_height": grid_h,
            "grid_data": {"format": "2d_array", "data": grid_data},
            "color_count": len(color_summary),
            "total_beads": total_beads,
            "color_summary": color_summary,
            "board_count": board_layout["total"] if board_layout else 1,
            "board_layout": board_layout,
            "estimated_time": self._estimate_time(total_beads),
            "difficulty": self._calc_difficulty(len(color_summary), total_beads),
            "preview_url": preview_url,
        }
    
    def _evaluate_suitability(self, image: np.ndarray) -> dict:
        """评估图片适合做拼豆的程度"""
        # 基于以下维度评分 (1-5):
        # - 色彩复杂度 (颜色越少越适合)
        # - 细节密度 (细节越少越适合)
        # - 对比度 (对比度越高越适合)
        # - 主体明确度 (主体越突出越适合)
        
        # 简化实现：统计颜色直方图复杂度
        unique_colors = len(np.unique(image.reshape(-1, 3), axis=0))
        
        if unique_colors < 1000:
            score = 5
            hint = "这张图颜色简洁，非常适合做拼豆"
        elif unique_colors < 5000:
            score = 4
            hint = "这张图主体突出，适合做拼豆"
        elif unique_colors < 20000:
            score = 3
            hint = "这张图细节适中，建议选择标准版或简单版"
        elif unique_colors < 50000:
            score = 2
            hint = "这张图细节较多，简单版可能丢失部分内容"
        else:
            score = 1
            hint = "这张图非常复杂，建议更换主体更突出的图"
        
        return {"score": score, "hint": hint}
    
    def _estimate_time(self, total_beads: int) -> int:
        """估算制作时间(分钟)，经验值：约15颗/分钟"""
        return max(10, round(total_beads / 15))
    
    def _calc_difficulty(self, color_count: int, total_beads: int) -> str:
        if color_count <= 8 and total_beads <= 400:
            return "easy"
        elif color_count <= 18 and total_beads <= 1500:
            return "medium"
        else:
            return "hard"
```

#### 4.3.1 GridRenderer — 网格图稿渲染器

```typescript
// canvas/GridRenderer.ts

interface GridRendererOptions {
  canvas: HTMLCanvasElement;
  gridData: string[][];         // 二维颜色ID数组
  paletteMap: Map<string, string>;  // colorId → hex
  cellSize: number;             // 每格像素大小
  showGrid: boolean;            // 是否显示网格线
  showColorCode: boolean;       // 是否显示色号文字
}

class GridRenderer {
  private ctx: CanvasRenderingContext2D;
  private gridData: string[][];
  private paletteMap: Map<string, string>;
  
  // 视口状态 (支持缩放和平移)
  private viewport = {
    offsetX: 0,
    offsetY: 0,
    scale: 1,
  };
  
  constructor(options: GridRendererOptions) { ... }
  
  /**
   * 渲染完整图稿
   * 性能优化：只渲染视口内可见的格子
   */
  render(): void {
    const ctx = this.ctx;
    ctx.clearRect(0, 0, this.canvas.width, this.canvas.height);
    
    ctx.save();
    ctx.translate(this.viewport.offsetX, this.viewport.offsetY);
    ctx.scale(this.viewport.scale, this.viewport.scale);
    
    // 计算可见范围
    const visibleBounds = this.getVisibleBounds();
    
    // 渲染格子
    for (let y = visibleBounds.startY; y < visibleBounds.endY; y++) {
      for (let x = visibleBounds.startX; x < visibleBounds.endX; x++) {
        this.renderCell(x, y);
      }
    }
    
    // 渲染网格线
    if (this.showGrid && this.viewport.scale > 0.5) {
      this.renderGridLines(visibleBounds);
    }
    
    // 渲染色号文字 (缩放足够大时)
    if (this.showColorCode && this.viewport.scale > 2) {
      this.renderColorCodes(visibleBounds);
    }
    
    ctx.restore();
  }
  
  /**
   * 高亮指定位置的格子 (跟做模式用)
   */
  renderHighlight(positions: [number, number][], color: string): void { ... }
  
  /**
   * 弱化非高亮区域 (半透明遮罩)
   */
  renderDimOverlay(excludePositions: [number, number][]): void { ... }
  
  /**
   * 屏幕坐标 → 网格坐标
   */
  screenToGrid(screenX: number, screenY: number): { x: number; y: number } | null { ... }
  
  /**
   * 设置视口 (缩放/平移)
   */
  setViewport(offsetX: number, offsetY: number, scale: number): void { ... }
  
  /**
   * 导出高清图片 (用于导出功能)
   */
  exportToImage(pixelPerCell: number = 20): HTMLCanvasElement { ... }
}
```

```typescript
  setActiveTool: (tool: ToolType) => void;
  setActiveColor: (colorId: string) => void;
  updateCell: (x: number, y: number, colorId: string) => void;
  updateCells: (cells: {x: number, y: number, colorId: string}[]) => void;
  replaceColor: (fromId: string, toId: string) => void;
  undo: () => void;
  redo: () => void;
  pushHistory: () => void;
  toggleGrid: () => void;
  markSaved: () => void;
}

export const useEditorStore = create<EditorState>((set, get) => ({
  gridData: null,
  gridWidth: 0,
  gridHeight: 0,
  activeTool: 'point',
  activeColorId: '',
  showGrid: true,
  showColorCode: false,
  showOriginalRef: false,
  history: [],
  historyIndex: -1,
  isDirty: false,
  paletteId: '',
  paletteColors: [],

  updateCell: (x, y, colorId) => {
    const { gridData } = get();
    if (!gridData) return;
    
    // 先推入历史
    get().pushHistory();
    
    const newGrid = gridData.map(row => [...row]);
    newGrid[y][x] = colorId;
    
    set({ gridData: newGrid, isDirty: true });
  },
  
  undo: () => {
    const { history, historyIndex } = get();
    if (historyIndex <= 0) return;
    
    const newIndex = historyIndex - 1;
    set({
      gridData: history[newIndex].map(row => [...row]),
      historyIndex: newIndex,
      isDirty: true,
    });
  },
  
  pushHistory: () => {
    const { gridData, history, historyIndex } = get();
    if (!gridData) return;
    
    const snapshot = gridData.map(row => [...row]);
    const newHistory = history.slice(0, historyIndex + 1);
    newHistory.push(snapshot);
    
    // 限制历史栈大小
    if (newHistory.length > 30) {
      newHistory.shift();
    }
    
    set({
      history: newHistory,
      historyIndex: newHistory.length - 1,
    });
  },
  
  // ... 其他 actions
}));
```

```typescript
function useAutoSave(workId: string, gridData: string[][] | null) {
  useEffect(() => {
    if (!workId || !gridData) return;
    
    // 每 30 秒自动保存到 IndexedDB
    const timer = setInterval(() => {
      saveToLocal(workId, gridData);
    }, 30000);
    
    // 页面关闭/切换时立即保存
    const handleVisibilityChange = () => {
      if (document.hidden) {
        saveToLocal(workId, gridData);
      }

    document.addEventListener('visibilitychange', handleVisibilityChange);
    
    return () => {
      clearInterval(timer);
      document.removeEventListener('visibilitychange', handleVisibilityChange);
    };
  }, [workId, gridData]);
}
```

```python
# utils/pdf.py (简化)

from PIL import Image, ImageDraw, ImageFont

class GridImageExporter:
    """生成图稿图片"""
    
    CELL_SIZE = 20           # 每格像素
    GRID_LINE_WIDTH = 1      # 网格线宽
    FONT_SIZE = 8            # 色号字体大小
    
    def export_color(
        self,
        grid_data: list[list[str]],
        palette_map: dict,      # color_id → hex
        color_names: dict,      # color_id → name
        work_name: str,
        show_grid: bool = True,
        show_codes: bool = True,
        watermark: bool = False,
    ) -> Image.Image:
        """导出彩色图稿图片"""
        
        h = len(grid_data)
        w = len(grid_data[0])
        
        # 画布尺寸
        img_w = w * self.CELL_SIZE + 1
        img_h = h * self.CELL_SIZE + 1
        
        # 加上信息区域
        info_height = 200  # 底部信息区
        total_h = img_h + info_height
        
        img = Image.new('RGB', (img_w, total_h), 'white')
        draw = ImageDraw.Draw(img)
        
        # 绘制格子
        for y in range(h):
            for x in range(w):
                color_id = grid_data[y][x]
                hex_color = palette_map.get(color_id, '#FFFFFF')
                
                x1 = x * self.CELL_SIZE
                y1 = y * self.CELL_SIZE
                x2 = x1 + self.CELL_SIZE
                y2 = y1 + self.CELL_SIZE
                
                draw.rectangle([x1, y1, x2, y2], fill=hex_color)
        
        # 绘制网格线
        if show_grid:
            for x in range(w + 1):
                draw.line(
                    [(x * self.CELL_SIZE, 0), (x * self.CELL_SIZE, img_h)],
                    fill='#CCCCCC', width=self.GRID_LINE_WIDTH
                )
            for y in range(h + 1):
                draw.line(
                    [(0, y * self.CELL_SIZE), (img_w, y * self.CELL_SIZE)],
                    fill='#CCCCCC', width=self.GRID_LINE_WIDTH
                )
        
        # 绘制色号文字
        if show_codes:
            font = ImageFont.truetype("path/to/font.ttf", self.FONT_SIZE)
            for y in range(h):
                for x in range(w):
                    color_id = grid_data[y][x]
                    cx = x * self.CELL_SIZE + self.CELL_SIZE // 2
                    cy = y * self.CELL_SIZE + self.CELL_SIZE // 2
                    
                    # 根据背景色选择黑/白文字
                    text_color = self._contrast_color(palette_map[color_id])
                    draw.text(
                        (cx, cy), color_id, fill=text_color,
                        font=font, anchor='mm'
                    )
        
        # 绘制底部信息区 (作品名、颜色图例、统计)
        self._draw_info_section(draw, img_h, img_w, grid_data, palette_map, color_names, work_name)
        
        # 水印
        if watermark:
            self._draw_watermark(draw, img_w, total_h)
        
        return img
```

```python
# core/security.py

from datetime import datetime, timedelta
from jose import jwt, JWTError
from passlib.context import CryptContext

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

class SecurityService:
    SECRET_KEY = config.SECRET_KEY
    ALGORITHM = "HS256"
    ACCESS_TOKEN_EXPIRE = timedelta(hours=24)
    REFRESH_TOKEN_EXPIRE = timedelta(days=30)
    
    @staticmethod
    def create_access_token(user_id: str, platform: str) -> str:
        payload = {
            "sub": user_id,
            "platform": platform,
            "exp": datetime.utcnow() + SecurityService.ACCESS_TOKEN_EXPIRE,
            "type": "access",
        }
        return jwt.encode(payload, SecurityService.SECRET_KEY, SecurityService.ALGORITHM)
    
    @staticmethod
    def verify_token(token: str) -> dict:
        try:
            payload = jwt.decode(token, SecurityService.SECRET_KEY, algorithms=[SecurityService.ALGORITHM])
            return payload
        except JWTError:
            raise UnauthorizedException("Token无效或已过期")
```

```typescript
class GridRenderer {
  /**
   * 性能优化：脏区域渲染
   * 只重绘发生变化的格子，不全量重绘
   */
  renderDirty(dirtyCells: Set<string>): void {
    for (const key of dirtyCells) {
      const [x, y] = key.split(',').map(Number);
      this.renderCell(x, y);
      if (this.showGrid) this.renderCellGrid(x, y);
    }
    dirtyCells.clear();
  }
  
  /**
   * 性能优化：分层 Canvas
   * - 底层：图稿内容 (不频繁变化)
   * - 中层：高亮/选中 (跟做模式频繁变化)  
   * - 顶层：交互指示器 (光标/选框，每帧变化)
   */
  private layers = {
    content: document.createElement('canvas'),
    highlight: document.createElement('canvas'),
    interaction: document.createElement('canvas'),
  };
}
```

**后端新增接口：**

```python
# api/v1/auth.py

@router.post("/wechat-login")
async def wechat_login(request: WechatLoginRequest):
    """微信小程序登录
    
    1. 前端调用 wx.login() 获取 code
    2. 后端用 code 换取 openid + session_key
    3. 查找或创建用户
    4. 返回 JWT Token
    """
    # 调用微信 API
    result = await wechat_service.code2session(request.code)
    openid = result["openid"]
    
    # 查找或创建用户
    user = await user_service.find_by_platform("wechat_mp", openid)
    if not user:
        user = await user_service.create(
            platform="wechat_mp",
            platform_openid=openid,
            nickname=request.nickname,
        )
    
    # 签发 Token
    token = SecurityService.create_access_token(user.id, "wechat_mp")
    return {"token": token, "user": user}
```
