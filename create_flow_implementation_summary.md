#### 交互优化

- **参数前置**：用户在上传前就能设置期望，避免转换后不满意重来
- **自动收起**：转换完成后参数区自动收起，节省空间
- **快速调整**：展开参数区可快速调整并重新转换
- **一键编辑**：点击图纸或按钮直接进入编辑模式
- **更换图片**：点击原图缩略图可快速更换图片

```typescript
interface ConversionParams {
  boardWidth: number;          // 板子宽度
  boardHeight: number;         // 板子高度
  colorBrand: ColorBrand;      // 色系品牌
  colorCount: number | 'auto'; // 颜色数量
  removeBackground: boolean;   // 去除背景
  enhanceContrast: boolean;    // 对比度增强
  optimizeEdges: boolean;      // 边缘优化
  removeNoise: boolean;        // 噪点去除
}

interface ConversionResult {
  originalImage: string;       // 原图
  convertedGrid: string[][];   // 转换后的网格数据
  preview: string;             // 预览图
  stats: {                     // 统计信息
    width: number;
    height: number;
    colors: number;
    beadCount: number;
    estimatedTime: string;
    estimatedCost?: string;
  };
}
```

## 设计亮点（V2）

### 1. 参数前置设计
- **用户痛点**：传统流程是先上传再选方案，如果不满意需要重新上传
- **解决方案**：参数前置，用户在上传前就能设置期望的尺寸、色系等
- **优势**：一次上传，精准转换，避免重复操作

### 2. 智能预设系统
- **新手模式**：小尺寸+少颜色，快速完成，降低门槛
- **标准模式**：中等尺寸+适中颜色，平衡效果和难度
- **专业模式**：大尺寸+丰富颜色，追求细节和质量
- **优势**：一键应用，降低决策成本

### 3. 品牌色系支持
- **智能选色**：自动选择最佳颜色，适合不确定的用户
- **品牌色系**：MARD/COCO/漫漫/Artkal，匹配用户已有豆豆
- **优势**：提升颜色匹配准确度，减少购买成本

### 4. 去除背景功能
- **用户痛点**：实物拍照背景杂乱，影响转换效果
- **解决方案**：智能识别主体，自动去除背景
- **优势**：提升转换质量，适合拍照场景

### 5. 参数可调整
- **转换后收起**：节省空间，突出图纸预览
- **展开调整**：不满意可快速调整参数重新转换
- **优势**：灵活性高，支持迭代优化

### 6. 一键进入编辑
- **点击图纸**：直接进入编辑模式
- **底部按钮**：明确的操作入口
- **优势**：流程连贯，减少跳转步骤
